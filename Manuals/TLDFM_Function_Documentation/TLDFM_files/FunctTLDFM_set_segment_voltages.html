TLDFM_set_segment_voltages

ViStatus TLDFM_set_segment_voltages (ViSession instrumentHandle, ViReal64 _VI_FAR segmentVoltages[]);

Purpose

This function sets the voltages of all segments.
Note: When the hysteresis compensation is switched on, the real physical voltages at the mirror will differ from 'set voltages' due to the mathematical hysteresis compensation.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
segmentVoltages ViReal64[] This parameter sets the voltage of all segments in volts.
The array size needs to fit the total number of segments.
For the total number of segments call function TLDFM_get_device_configuration or TLDFM_get_segment_count.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS             Get Segment Voltages operation successful
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
