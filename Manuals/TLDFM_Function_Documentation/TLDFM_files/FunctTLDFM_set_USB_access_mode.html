TLDFM_set_USB_access_mode

ViStatus TLDFM_set_USB_access_mode (ViSession instrumentHandle, ViUInt32 accessMode, ViString requestedKey, ViChar _VI_FAR accessKey[]);

Purpose

This function is used to obtain a lock on the deformable mirror instrument. This operation can also be used for sharing and nesting locks.

Parameters

Input
Name Type Description
instrumentHandle ViSession
accessMode ViUInt32 This parameter determines how the access to the instrument should be controlled.
Available settings:
0  AM_No_Lock          free access for other instances
1  AM_Exclusive_Lock   lock the device to the current instance
2  AM_Shared_Lock      shared look with other instances
requestedKey ViString This string parameter specifies the requested key and is used only when requesting a shared lock.  If the operation is successful, the key can be passed to other sessions for lock sharing. If not specified, VISA will generate a unique key.
Output
Name Type Description
accessKey ViChar[] The actual access key to the resource associated with the Instrument Handle session.  Other sessions can pass this key as the RequestedKey to share the lock on the resource.
If the lock being requested is an exclusive lock, this parameter can be VI_NULL.
Note:  String declaration size must be at least 256 bytes.

Return Value

Name Type Description
status ViStatus
