TLDFM_get_measured_tilt_voltages

ViStatus TLDFM_get_measured_tilt_voltages (ViSession instrumentHandle, ViReal64 _VI_FAR tiltVoltages[]);

Purpose

This function returns the voltages of all tilt elements.
Note: When the hysteresis compensation is switched on this functions will read back real physical voltages at the mirror. These may differ from written 'set voltages' due to the mathematical hysteresis compensation.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Output
Name Type Description
tiltVoltages ViReal64[] This parameter returns the voltage of all tilt elements in volts.
The array size needs to fit the total number of tilt elements.
For the total number of tilt elements call function TLDFM_get_device_configuration or TLDFM_get_tilt_count.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS             Get Segment Voltages operation successful
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
