TLDFM_get_hysteresis_parameters

ViStatus TLDFM_get_hysteresis_parameters (ViSession instrumentHandle, ViUInt32 target, ViPUInt32 count, ViPReal64 nonlinearFactor, ViReal64 _VI_FAR arrayTresholdInverter[], ViReal64 _VI_FAR arrayWeightInverter[]);

Purpose

This function returns hysteresis compensation parameters of the instrument.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
target ViUInt32 This parameter determines the target item.
Valid values:
0   T_MIRROR
1   T_TILT
Output
Name Type Description
count ViUInt32 (passed by reference) This parameter returns the number of available hysteresis sample points.
nonlinearFactor ViReal64 (passed by reference) This parameter returns the nonlinear factor used to correct the hysteresis curve.
arrayTresholdInverter ViReal64[] This array returns Count numbers of inverse treshold values used to compensate for the hysteresis.
arrayWeightInverter ViReal64[] This array returns Count numbers of inverse weight values used to compensate for the hysteresis.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS          Get the instrument name successful.
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
