TLDFMX_set_voltages_setpoint

ViStatus TLDFMX_set_voltages_setpoint (ViSession instrumentHandle, ViReal64 _VI_FAR mirrorVoltages[]);

Purpose

This function is used to define new segment voltages to the Deformable Mirror under the consideration that already applied Zernike voltage pattern should be taken into account and not being overwritten.
This function just updates the TLDFMX internal driver data so that the voltage setpoints can be taken into account by following function calls to TLDFMX_calculate_single_zernike_pattern or TLDFMX_calculate_zernike_pattern.
The goal is to manage user requests for setting a certain Zernike pattern AND at the same time fulfill requests to set one or more segments to a defined voltage.
Both demands can be met by
- Calling TLDFMX_calculate_single_zernike_pattern or TLDFMX_calculate_zernike_pattern to set new Zernke amplitudes. All segments may change its voltage at once.
- Calling TLDFMX_set_single_voltage_setpoint or TLDFMX_set_voltages_setpoint to define segment voltages under consideration of already present Zernike pattern. Subsequently, function TLDFMX_calculate_single_zernike_pattern or TLDFMX_calculate_zernike_pattern needs to be called to get the combined pattern.
Note:
The desired mirror setpoint voltages have to be set separately to the physical device by using TLDFM base driver functions like TLDFM_set_segment_voltages.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
mirrorVoltages ViReal64[] This parameter accepts the array of the new voltage setpoints in volts.
Array Size: at least (total no. of segments) elements
Valid range: (Segment Voltage Min) ... (Segment Voltage Max)
Call base driver function TLDFM_get_device_configuration to get the valid voltage range and the total number of segments.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS              Update successful
