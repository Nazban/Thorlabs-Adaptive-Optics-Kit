TLDFMX_get_system_parameters

ViStatus TLDFMX_get_system_parameters (ViSession instrumentHandle, TLDFMX_rotation_t *rotationMode, TLDFMX_flip_t *flipMode, ViReal64 _VI_FAR maxDeviceZernikeAmplitudes[], ViPBoolean isDataValid);

Purpose

This function returns parameters of the optical setup measured by the function TLDFMX_measure_system_parameters.
Determined parameters are
- the required rotation and flip option for the Zernike pattern
- the maximum achievable Zernike amplitudes.
Note:
(1) A successful run of function TLDFMX_measure_system_parameters is a precondition to get the measured data out herewith.
(2) You need to call functions TLDFMX_set_rotation_mode and TLDFMX_set_flip_mode in order to set the required options active.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Output
Name Type Description
rotationMode TLDFMX_rotation_t (passed by reference) This parameter returns the required rotation mode for the Zernike pattern generation.
Note:
You need to call functions TLDFMX_set_rotation_mode in order to set the required option active.
flipMode TLDFMX_flip_t (passed by reference) This parameter returns the required flip mode for the Zernike pattern generation.
Note:
You need to call functions TLDFMX_set_flip_mode in order to set the required option active.
maxDeviceZernikeAmplitudes ViReal64[] This parameter returns an array of Zernike amplitudes which are achievable by the DMP40 within the optical setup. The unit equals the unit of the Wavefront Sensor data feed into function TLDFMX_measure_system_parameters. By default this is microns.
The measured amplitude denotes the entire range so that minimum and maximum amplitudes are +/- the half range.
e.g.
amplitude_range = 5.2
amplitude_min = -amplitude_range / 2.0;
amplitude_max =  amplitude_range / 2.0;
Note:
(1) The required array size can be queried by function TLDFMX_get_parameters, parameter 'Maximum Zernike Count'.
(2) The array is zero based, that is index [0] denotes the Z4 amplitude and index [11] the Z15 amplitude.
(3) The measured amplitudes depend on the magnification of the optical setup.
isDataValid ViBoolean (passed by reference) This parameter returns if there are valid measured system parameters.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS              Close successful
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
