TLDFMX_set_rotation_mode

ViStatus TLDFMX_set_rotation_mode (ViSession instrumentHandle, ViUInt32 rotationMode);

Purpose

This function is used to adapt the Zernike amplitudes, generated by the Deformable Mirror (using functions TLDFMX_calculate_single_zernike_pattern and TLDFMX_calculate_zernike_pattern) to the Zernike amplitudes measured by a Wavefront Sensor.
If the mounting of these components within the optical setup is not in default position, this will result in a rotated wavefront. This is compensated by rotating the voltage pattern of the Deformable Mirror in counter clockwise direction.
Use functions TLDFMX_measure_system_parameters and TLDFMX_get_system_parameters to determine the correct rotation mode setting automatically.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
rotationMode ViUInt32 This parameter defines the rotation mode setting for generating Zernike pattern.
Available options:
R_None_Flag   = 0x00000000, //   0° ccw
R_45Deg_Flag  = 0x00000001, //  45° ccw
R_90Deg_Flag  = 0x00000002, //  90° ccw
R_135Deg_Flag = 0x00000003, // 135° ccw
To determine the correct setting for the setup use functions TLDFMX_measure_system_parameters and TLDFMX_get_system_parameters.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS              Close successful
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
