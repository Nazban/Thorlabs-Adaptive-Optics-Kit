TLDFMX_get_parameters

ViStatus TLDFMX_get_parameters (ViSession instrumentHandle, ViPReal64 minimumZernikeAmplitude, ViPReal64 maximumZernikeAmplitude, ViPInt32 maximumZernikeCount, int *measurementSteps, int *relaxSteps);

Purpose

This function returns important parameters for using TLDFMX driver functions.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Output
Name Type Description
minimumZernikeAmplitude ViReal64 (passed by reference) This parameter returns the minimum adjustable Zernike amplitude on a relative scale.
Default: -1.0
Note: You may pass VI_NULL if this information is not required.
maximumZernikeAmplitude ViReal64 (passed by reference) This parameter returns the maximum adjustable Zernike amplitude on a relative scale.
Default: +1.0
Note: You may pass VI_NULL if this information is not required.
maximumZernikeCount ViInt32 (passed by reference) This parameter returns the number of adjustable Zernike amplitudes. This also determines the minimum size of Zernike amplitude arrays.
Default: 12
Note:
(1) Zernike amplitude arrays range from index [0] for Z4 to index [11] for Z15.
(2) You may pass VI_NULL if this information is not required.
measurementSteps int (passed by reference) This parameter returns the necessary step count for measuring the optical system parameters in function TLDFMX_measure_system_parameters.
Default: 33
Note: You may pass VI_NULL if this information is not required.
relaxSteps int (passed by reference) This parameter returns the necessary step count for relaxing the mirror in function TLDFMX_relax.
Default: 80
Note: You may pass VI_NULL if this information is not required.

Return Value

Name Type Description
status ViStatus Operational return status. Contains either a completion code or an error code. Instrument driver specific codes that may be returned in addition to the VISA error codes defined in VPP-4.3 and vendor specific codes, are as follows.
Completition Codes
----------------------------------------------------------------
VI_SUCCESS              Get successful
Vendor Specific Codes
----------------------------------------------------------------
For error codes and descriptions see <Error Message>.
